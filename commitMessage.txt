
Github에 올리실 때, commit 메세지를 통일해주시면 감사하겠습니다!!

다름이 아니라 프로젝트시 커밋 연습을 위해서 하는겁니다.

커밋 메세지

풀었을 경우 -> solve : [플랫폼이름] {문제이름}_{날짜}
못푼 경우 -> unsolve : [플랫폼이름] {문제이름}_{날짜}

ex) solve : [BOJ] 다리놓기_240703
ex) unsolve : [PRO] 피로도_240705

마지막 커밋 메세지로 위 내용이 남아있으면 됩니다.
플랫폼 표기 
BOJ -> 백준
PRO -> 프로그래머스
CDT -> 코드트리
이후 추가되는 플랫폼은 업데이트 하겠습니다.
